Breadcrumbs

Indicate the current page's location within a navigational hierarchy.

For Modern browser (not ie6, 7), Separators are automatically added in CSS through :before and content.

Default Breadcrumb

  1. Home
  2. Item 1
  3. Item 2
  4. Item 3
  5. Item 4
  6. Item 5
<ol class="gui-breadcrumb">
  <li><a href="#">Home</a></li>
  <li><a href="#">Item 1</a></li>
  <li><a href="#">Item 2</a></li>
  <li><a href="#">Item 3</a></li>
  <li><a href="#">Item 4</a></li>
  <li class="active">Item 5</li>
 </ol>

If you want it works on ie6, 7 or you need the behavior (remove siblings when click event triggered). use following codes.

$(".gui-breadcrumb").guiBreadcrumb();
