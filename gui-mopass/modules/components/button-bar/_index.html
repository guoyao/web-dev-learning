Button bar

Group a series of buttons together on a single line with the button bar. Add on optional JavaScript radio and checkbox style behavior with our button bar plugin.

<div class="gui-btn-bar">
 <button type="button" class="gui-btn gui-btn-default">Left</button>
 <button type="button" class="gui-btn gui-btn-default">Middle</button>
 <button type="button" class="gui-btn gui-btn-default">Right</button>
</div>

Toggle Button bar

Checkbox

Add data-toggle="buttons" to a group of checkboxes for checkbox style toggling on gui-btn-bar.

<div class="gui-btn-bar" data-toggle="buttons">
  <label class="gui-btn gui-btn-primary">
    <input type="checkbox"> Option 1
  </label>
  <label class="gui-btn gui-btn-primary">
    <input type="checkbox"> Option 2
  </label>
  <label class="gui-btn gui-btn-primary">
    <input type="checkbox"> Option 3
  </label>
</div>

Radio

Add data-toggle="buttons" to a group of radio inputs for radio style toggling on gui-btn-bar.

<div class="gui-btn-bar" data-toggle="buttons">
  <label class="gui-btn gui-btn-primary">
    <input type="radio" name="options" id="option1"> Option 1
  </label>
  <label class="gui-btn gui-btn-primary">
    <input type="radio" name="options" id="option2"> Option 2
  </label>
  <label class="gui-btn gui-btn-primary">
    <input type="radio" name="options" id="option3"> Option 3
  </label>
</div>
