Graceful Web UI简介

  • Graceful-web-ui 是一个web前端开发框架的开源项目，目的是创建一个优雅、跨浏览器、直观、强大的web ui framework，从而加快web前端开发，使得开发更加容易。
  • 使用了Less、Nodejs、Bower、Qunit、Normalize等， 借鉴了Bootstrap的部分思想；各个组件依赖于Jquery，以Jquery Plugin的形式开发。
  • 使用Git来实现版本控制，使用Github来管理项目、协作开发。 项目地址：https://github.com/guoyao/graceful-web-ui

项目整体结构

  • assets目录存放项目需要的image、icon等资源文件。
  • bower_components、dist、
    node_modules 都是临时目录，每次编译时都会自动生成。分别存放项目依赖、编译的最终结果、自动化编译/测试等所需的nodejs模块。
  • demo目录存放demo、文档等。
  • js目录存放项目所有的javascript文件，在编译时会被合并。
  • less目录存放项目所有的less文件，编译时会先合并，然后翻译成css。

项目整体结构

  • bower.json文件存放项目所需要的第三方依赖，包括jquery、qunit等。
  • Gruntfile.js文件存放项目自动化编译、合并、打包等脚本。
  • package.json文件存放项目的描述文件，所需的nodejs模块依赖等。
